## Project Overview
The real estate market is complex, and pricing a house correctly can be difficult. The goal of this project is to build a machine learning model that can estimate a house's selling price. By doing this, we also figure out exactly which features (like having an AC or an extra bathroom) drive the price up the most!

## Dataset Information
The dataset used in this project contains **545 records** of houses with **13 features**:
- **Target Variable:** `price` (The actual selling price of the house)
- **Numerical Features:** `area`, `bedrooms`, `bathrooms`, `stories`, `parking`
- **Categorical Features:** `mainroad`, `guestroom`, `basement`, `hotwaterheating`, `airconditioning`, `prefarea`, `furnishingstatus`
